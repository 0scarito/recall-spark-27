ICONS REQUIRED
==============

Please add the following icon files to this folder:

- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

QUICK FIX:
If you don't have these icons, you can:
1. Use any square PNG image you have
2. Copy it 4 times and rename them to the names above
3. Chrome will automatically resize them

The extension will not load without at least one icon file.
